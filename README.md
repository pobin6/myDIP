# myDIP
数字图像处理软件（Digital Image Processing soft） 基本的图像处理方法
目前有几乎大部分的空间对图像的操作。
请使用./myDIP/MyDIP/MyDIP/bin/Debug/MyDIP.exe运行软件
参数bitmapOrigin与bitmapO2为输入，选择需要处理的图片，
参数bitmapResult为输出，不用理会。
其他参数根据选中功能不同有不同的参数
最后，本软件存在bug可能较多，注意下。
